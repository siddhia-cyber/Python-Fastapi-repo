# RAG Backend with FastAPI, Qdrant & Ollama

This project implements a Retrieval-Augmented Generation (RAG) backen using:

- FastAPI for APIs
- Qdrant Cloud as a vector database
- Ollama for local text embeddings
- Chunking-based ingestion for large documents


## Features

- Store single texts as vector embeddings
- Store large documents using chunking
- Semantic search using cosine similarity
- Cloud-hosted vector storage (Qdrant)
- Local embeddings via Ollama
- fastapi 

# Tech Stack

- Python 3.12
- FastAPI
- Qdrant Cloud
- Ollama (`nomic-embed-text`)
- Requests
- python-dotenv
